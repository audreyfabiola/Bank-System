# Bank System
This is an online banking system created using python.

## Features
You can choose between two roles, administrator or customer.
### Administrator
- Add customer
- Delete customer
- Edit customer name, pin, and balance
- Search customer

### Customer
- Deposit money
- Withdraw money
- Check balance
